### 1.2 区域顺序

```
【游戏中的区域选择】

每局游戏会随机选择3-4个区域：

第一区域（必定）：回响森林
- 作为新手区域，难度最低
- 帮助玩家熟悉游戏机制

中间区域（随机1-2个）：熔岩/深海/骸骨
- 根据随机结果选择1-2个
- 每个都有独特的挑战

最终区域（必定）：深渊核心
- BOSS战所在地
- 真相揭露
- 最终选择
```

## 2. 区域一：回响森林

```
═══════════════════════════════════════════════════════════════
【回响森林 - Echo Forest】
═══════════════════════════════════════════════════════════════

位置：深渊第一层（最接近地表）
危险等级：★☆☆☆☆
推荐等级：Lv.1 - Lv.3

───────────────────────────────────────────────────────────────
【区域背景】
───────────────────────────────────────────────────────────────

"在世界的边缘，有一个永不沉寂的森林。
这里的每一棵树都在歌唱，
每一阵风都在低语，
每一片叶子都承载着一个声音的碎片。

但那曾经是美好的时光。
现在，那些歌声变得扭曲，
树木变成了陷阱，
动物变成了怪物。

只有森林最深处的那个声音，
依然保持着纯净。"

                                    ——聆听者协会·区域志·第一卷

───────────────────────────────────────────────────────────────
【环境描述】
───────────────────────────────────────────────────────────────

森林曾经是一片生机勃勃的地方。

在高大的古树之间，
生活着会唱歌的精灵，
他们的声音让万物生长。

但深渊的回响改变了一切。

树木枯萎了，
但它们的年轮中记录着无数的声音碎片。
树叶变成了锋利的刀刃，
专门切割那些还在唱歌的生命。

最可怕的是那些动物。
它们曾经是森林最动听的声音，
现在却变成了最危险的怪物。

但在森林最深处，
有一个古老的树洞，
那里保存着森林最纯净的回响——
森林之灵的声音。

───────────────────────────────────────────────────────────────
【物理特征】
───────────────────────────────────────────────────────────────

光线：
- 昏暗的绿色光芒
- 树冠间透下微弱的光线
- 发光苔藓提供环境光
- 某些区域完全黑暗

地形：
- 巨大的树干和树根作为掩体
- 地面覆盖着厚厚的落叶
- 偶尔有倒下的树木
- 存在藤蔓障碍

危险元素：
- 落叶陷阱（踩上去会尖叫，吸引敌人）
- 枯萎藤蔓（会缠绕玩家）
- 剧毒孢子（漂浮在空中）

───────────────────────────────────────────────────────────────
【声音特征】
───────────────────────────────────────────────────────────────

背景音：
- 扭曲的鸟鸣（忽远忽近）
- 风吹过枯叶的沙沙声
- 远处传来的脚步声
- 偶尔的尖叫

特殊声音：
- 树灵的歌声（在最深处）
- 精灵的悲叹（在古老遗迹中）
- 树根移动的声音（警戒信号）

互动声音：
- 采集回响碎片：清脆的叮咚声
- 敌人发现玩家：尖锐的啸叫
- 安全区域：平静的森林音乐

───────────────────────────────────────────────────────────────
【主要生物】
───────────────────────────────────────────────────────────────

【已感染的回响蝙蝠】
森林中的飞行单位，
原本是唱歌最动听的小生物。
现在它们的翅膀变成了声波刀刃。

【扭曲的爬行者】
原本在树上生活的小动物，
现在变成潜伏在阴影中的猎手。
它们的移动会发出回响波纹。

【森林之声猎手】
原本是森林精灵的守护者，
现在变成了森林的审判者。
它们会惩罚任何还在"唱歌"的存在。

【树灵守护者】
森林最古老的存在，
虽然被深渊影响，但保持着部分神智。
它们在等待有人能让森林重新歌唱。

───────────────────────────────────────────────────────────────
【BOSS：回响守卒】
───────────────────────────────────────────────────────────────

回响守卒曾经是森林的守护神，
用歌声维持着森林的平衡。

但在深渊的侵蚀下，
它变成了一个只知道消灭"入侵者"的机器。

战斗场景：
- BOSS房间是一个巨大的树洞
- 周围有无数发光的回响碎片
- 树洞墙壁上刻着古老的符文

战斗音乐：
- 低沉的鼓点
- 不和谐的和声
- 逐渐加速的节奏

战斗阶段：

第一阶段：
- 缓慢移动
- 发射声波弹幕
- 召唤回响蝙蝠

第二阶段（HP<50%）：
- 护甲碎裂
- 移动速度加快
- 地面冲击波

击败后：
- 守卒恢复了部分神智
- 说出森林的真相
- 森林中开出了一朵纯净的花

───────────────────────────────────────────────────────────────
【剧情碎片位置】
───────────────────────────────────────────────────────────────

1. 森林入口：
   "欢迎来到回响森林。
    这里曾经是歌唱者的天堂。
    现在……请小心你的脚步。
    不要让它们听到你。"

2. 古树遗迹：
   "这些树已经有千年的历史。
    它们的年轮里，记录着无数的故事。
    用你的眼睛，去读取它们的声音。"

3. 精灵废墟：
   "这里曾经住着森林的精灵。
    他们的歌声让万物生长。
    现在他们已经消逝，
    但他们的声音留在了这里。"

4. 守卒房间：
   "我是回响守卒。
    我曾经是这片森林的守护者。
    但深渊改变了我。
    让我只看到了入侵者……
    谢谢你解放了我。"

───────────────────────────────────────────────────────────────
【可发现物品】
───────────────────────────────────────────────────────────────

【回响碎片】
- 森林中的回响碎片呈绿色
- 从发光的苔藓中收集
- 包含森林生物的记忆

【森林之心】
- 击败BOSS后获得
- 永久Buff：生命恢复速度+10%

【精灵之歌】
- 特殊道具
- 使用后可以短暂听到森林精灵的声音

【歌声残响】
- 收集全部剧情碎片后获得
- 解锁艾琳娜的第一段回忆
```

## 3. 区域二：熔岩回响

```
═══════════════════════════════════════════════════════════════
【熔岩回响 - Lava Echo】
═══════════════════════════════════════════════════════════════

位置：深渊第二层（地下深处）
危险等级：★★★☆☆
推荐等级：Lv.3 - Lv.5

───────────────────────────────────────────────────────────────
【区域背景】
───────────────────────────────────────────────────────────────

"在最深的地底，温度即是频率。
愤怒的声音在这里燃烧得最旺，
恐惧的声音早已汽化消失。

这里埋葬着无数未能说出的话语，
它们在地狱之火中永远回响。

如果你能在这里找到寂静，
那你就能找到真正的勇气。"

                                    ——聆听者协会·区域志·第二卷

───────────────────────────────────────────────────────────────
【环境描述】
───────────────────────────────────────────────────────────────

熔岩回响是深渊中温度最高的区域。

这里的地形由凝固的熔岩和永不熄灭的火焰构成。
回响在这里不会消逝，而是被火焰"保存"下来。

最常见的声音是愤怒的呐喊——
那些临终前最激烈的情感，
在这里被无限放大。

这片区域曾经是喧嚣者托尔的守护范围。
他的战鼓声至今仍在回响，
成为了这个区域的背景音乐。

但在战鼓声中，也混杂着其他声音：
那是那些没能逃出这片区域的聆听者……

───────────────────────────────────────────────────────────────
【物理特征】
───────────────────────────────────────────────────────────────

光线：
- 橙红色的火光
- 岩浆的微光
- 某些洞穴完全黑暗
- 火焰喷射时短暂变亮

地形：
- 岩浆河流和湖泊
- 凝固的岩石平台
- 需要跳跃的间隔
- 摇晃的石桥

危险元素：
- 岩浆（持续伤害）
- 火焰喷射（定时触发）
- 蒸汽爆炸（随机触发）
- 落石（攻击后触发）

温度系统：
- 靠近岩浆会受到持续火焰伤害
- 站在安全平台可以减免
- 某些技能在高温下威力提升

───────────────────────────────────────────────────────────────
【声音特征】
───────────────────────────────────────────────────────────────

背景音：
- 岩浆冒泡
- 火焰噼啪
- 高温嘶嘶声
- 战鼓的轰鸣（托尔）

特殊声音：
- 托尔的战鼓（随机出现）
- 受刑者的尖叫（在深处）
- 熔岩流动（环境背景）

互动声音：
- 获得护盾：火焰熄灭
- 火焰喷射：警报音
- 收集回响：燃烧的声音变成歌声

───────────────────────────────────────────────────────────────
【主要生物】
───────────────────────────────────────────────────────────────

【回响甲虫】
被高温改造成坚硬甲壳的甲虫。
它们在熔岩中生活，以回响为食。
攻击方式：滚动冲撞

【烈焰猎犬】
原本是普通的工作犬，
在熔岩中被改造成火焰生物。
它们的叫声是炽热的咆哮。

【熔岩巨像】
巨大的岩石生物，
由凝固的熔岩构成。
它们是熔岩层的守卫，
会用重拳攻击入侵者。

【托尔的声音残响】
虽然不是敌人，
但托尔的战鼓声会在战斗中为玩家提供增益。
找到托尔可以解锁他的回忆。

───────────────────────────────────────────────────────────────
【BOSS：颤音魔女】
───────────────────────────────────────────────────────────────

颤音魔女原本是一位歌手，
她的声音可以控制火焰。

在深渊中，她的声音被扭曲，
变成了控制回响的能力。

她现在在熔岩层深处，
用火焰和回响的力量，
惩罚所有"污染"深渊的入侵者。

战斗场景：
- 巨大的火山口
- 岩浆在脚下翻涌
- 悬浮的平台作为战斗场地
- 火焰形成的屏障

战斗音乐：
- 诡异的女声吟唱
- 低沉的弦乐
- 偶尔的高音尖叫

战斗阶段：

第一阶段：
- 悬浮在空中
- 发射火球和音符弹幕
- 放置火焰陷阱

第二阶段（HP<66%）：
- 分裂成两个分身
- 分身各有独立AI
- 分身死亡后合并
- 合并时造成范围伤害

第三阶段（HP<33%）：
- 召唤火焰风暴
- 地面变红区域需要躲避
- 移动速度大幅提升
- 偶尔隐身

击败后：
- 颤音魔女恢复了理智
- 唱出最后一首歌
- 岩浆逐渐冷却
- 托尔的战鼓声可以更清晰地听到

───────────────────────────────────────────────────────────────
【剧情碎片位置】
───────────────────────────────────────────────────────────────

1. 入口熔岩池：
   "这片熔岩中埋葬着无数的声音。
    每一个都是一段未完成的故事。
    那些愤怒的呐喊，
    那些恐惧的尖叫，
    都在这里永远回响。"

2. 托尔的战鼓台：
   "咚——咚——咚——
    托尔前辈的战鼓还在回响。
    即使他已经消逝，
    他的声音仍在守护着这片区域。"

3. 受刑者洞穴：
   "这里关押着那些犯下罪行的聆听者。
    他们的尖叫声永远不会停止。
    ……但他们的声音里，有罪的只是少数。"

4. 颤音魔女房间：
   "你来了……又一个入侵者。
    但你和他们不一样。
    我能听到你声音里的悲伤。
    和我一样……你也在寻找失去的人。"

───────────────────────────────────────────────────────────────
【可发现物品】
───────────────────────────────────────────────────────────────

【回响碎片】
- 熔岩中的回响碎片呈橙红色
- 从火焰中抢救出来
- 包含愤怒和恐惧的情感

【火焰之心】
- 击败BOSS后获得
- 永久Buff：攻击力+5%

【战鼓轰鸣】
- 在托尔的位置获得
- 解锁喧嚣者托尔作为可玩角色

【熔岩意志】
- 收集全部剧情碎片后获得
- 解锁艾琳娜的第二段回忆
```

## 4. 区域四：骸骨圣殿

```
═══════════════════════════════════════════════════════════════
【骸骨圣殿 - Bone Cathedral】
═══════════════════════════════════════════════════════════════

位置：深渊第四层（深层区域）
危险等级：★★★★☆
推荐等级：Lv.6 - Lv.8

───────────────────────────────────────────────────────────────
【区域背景】
───────────────────────────────────────────────────────────────

"在深渊的深处，有一个声音的墓地。
每一个未能说完的故事，
每一句未能传达的话语，
都在这里找到了永恒的归宿。

死者会在这里短暂复活，
那不是诅咒，而是祝福——
是深渊给予他们最后一次说话的机会。

但只有真正的聆听者，
才能听清他们想说的话。"

                                    ——聆听者协会·区域志·第四卷

───────────────────────────────────────────────────────────────
【环境描述】
───────────────────────────────────────────────────────────────

骸骨圣殿是深渊中最诡异的区域。

这里的一切都是由骨头构成的：
骨塔、骨桥、骨墙、骨地板。
每一块骨头都是某个消逝生命的一部分。

最奇特的是这里的时间规则。
死者会在这里复活——
但不是真正的复活，
而是他们的"声音"获得了短暂的实体。

在这个区域，死亡不是终结。
有时候，死亡意味着更多的战斗；
有时候，死亡意味着真相的揭露。

这里也是聆听者协会历代成员的归宿。
许多协会成员牺牲后，
他们的声音都会来到这里，
成为守护深渊入口的存在。

───────────────────────────────────────────────────────────────
【物理特征】
───────────────────────────────────────────────────────────────

光线：
- 幽暗的蓝色光芒
- 骨头本身的微光
- 偶尔有鬼火飘过
- 死者复活时短暂变亮

地形：
- 骨头构成的建筑遗迹
- 连接各区域的骨桥
- 漂浮的骨头平台
- 骨塔内部的隐藏房间

危险元素：
- 骨头陷阱（随机合拢）
- 灵魂漩涡（造成减速）
- 骨刺（从地面升起）
- 亡者复活（敌人复活）

复活系统：
- 某些敌人会复活一次
- 击杀后等待2秒，然后复活
- 再次击杀后彻底死亡
- BOSS不复活

───────────────────────────────────────────────────────────────
【声音特征】
───────────────────────────────────────────────────────────────

背景音：
- 骨头碰撞的咯咯声
- 幽冥的低语
- 远古的哭泣
- 死者的歌谣

特殊声音：
- 历代聆听者的合唱（固定地点）
- 开门声（骨头移动）
- 复活的声音（咯咯声变成尖叫）

互动声音：
- 击败亡者：安息之歌
- 收集回响：灵魂叹息
- 听到合唱：圣歌响起

───────────────────────────────────────────────────────────────
【主要生物】
───────────────────────────────────────────────────────────────

【回响骷髅】
被回响激活的骸骨战士。
它们曾经是真正的战士，
现在只保留着战斗的本能。

【合唱团领唱】
一群骷髅组成的合唱团。
它们会唱出死亡之歌，
```

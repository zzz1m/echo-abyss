## 1. 技能系统概述

### 1.1 技能分类

```
【技能类型】

┌─────────────────────────────────────────────────────────────┐
│                    技能分类体系                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  按资源消耗分类：                                            │
│  ├── 被动技能（无需操作，持续生效）                         │

│  └── 终极技能（消耗100能量，冷却较长）                      │
│                                                             │
│  按作用范围分类：                                            │
│  ├── 单体技能（作用于单个目标）                            │
│  ├── 范围技能（作用于区域）                                │
│  └── 辅助技能（作用于自身或友军）                          │

│  按功能分类：                                                │
│  ├── 伤害技能                                              │

│  ├── 恢复技能                                              │
│  └── 增益技能                                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 技能操作

```
【快捷键设置】

Q - 技能1
W - 技能2
E - 技能3
R - 终极技能（需要100能量）

空格 / Shift - 闪避
鼠标左键 - 基础攻击
鼠标右键 - 聆听技能（按住）

【操作模式】

1. 即时释放
   - 按下快捷键立即释放
   - 适用于QWE技能

2. 瞄准释放
   - 按下快捷键后需要瞄准
   - 显示技能范围指示
   - 适用于需要方向的技能

3. 持续施法
   - 按住技能键持续释放
   - 松开结束
   - 适用于范围治疗等
```

### 1.3 回响能量系统

```
【能量机制】

回响能量（Echo Energy）：
- 最大值：100点
- 初始值：100点
- 恢复方式：
  1. 时间恢复：每秒恢复3点
  2. 击杀恢复：击杀敌人恢复10点
  3. 受伤恢复：受到伤害恢复5点（冷却5秒）
  4. 特殊Buff

【能量显示】

屏幕左下角有回响能量条：
- 颜色：青色（#08D9D6）
- 满能量时有发光效果
- 释放终极技能时有爆发特效
```

```

═══════════════════════════════════════════════════════════════

- 升级可增强效果或添加新效果

【技能重置】

- 篝火处可免费重置技能
- 区域BOSS房前可重置
- 次数不限
```

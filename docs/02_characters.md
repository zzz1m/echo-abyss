## 1. 可玩角色

### 1.1 默认角色：聆听者艾文

```
═══════════════════════════════════════════════════════════════
【主角 - 最后的聆听者】
═══════════════════════════════════════════════════════════════

姓名：艾文（Evan）[玩家可自定义]
身份：最后的见习聆听者
年龄：20岁
身高：175cm

【外貌描述】

艾文是一个年轻的男性，
有着略显苍白的面庞和深棕色的短发。

他最显著的特征是眼睛上的眼罩——
这不是普通的眼罩，
而是聆听者协会传承的"聆听者之眼"。

戴上这个眼罩后，
艾文看不到普通的光线，
但能"看到"声音的形态。

眼罩下方偶尔会透出淡淡的青色光芒，
那是回响能量的颜色。

服装：
- 深紫色的连帽斗篷（协会标准服装）
- 斗篷边缘有若隐若现的声波纹路
- 脖子上挂着协会的聆听者徽章
- 腰间系着一个小型回响收集器

【性格特点】

1. 内敛但坚定
   艾文不是一个话多的人，
   但他内心有着不可动摇的决心。

2. 善于倾听
   作为聆听者，他习惯先听后说，
   这种习惯也延伸到了人际交往中。

3. 背负愧疚
   他认为自己应该和其他人一起牺牲，
   而不是独自存活。

4. 成长型
   游戏过程中，艾文会逐渐从迷茫走向坚定，
   最终找到自己的道路。

【背景故事】

艾文出生在一个被深渊污染的村庄。
在他五岁那年，
一次深渊潮汐摧毁了他的家园。

他是那场灾难中唯一的幸存者，
被大聆听者艾琳娜发现并收养。

在协会中长大，
艾文接受了聆听者的训练，
但由于年纪较小，
他始终只是见习聆听者。

两年前，当协会遭受毁灭性打击时，
艾琳娜让他先行撤退，
独自留下来断后。

从那以后，
艾文一直活在愧疚和自责中。
直到他发现了艾琳娜留下的信息，
他才决定重返深渊。

【内心独白】

"我常常想，如果当时留下来的是我……
不，不应该这样想。
艾琳娜姐姐说过，
活着本身就是一种使命。
我必须……让她的牺牲有意义。"

【战斗风格】

艾文是一个平衡型角色：
- 中等血量
- 中等伤害
- 中等速度
- 优秀的辅助能力

他擅长：
- 利用声波进行中距离攻击
- 感知敌人的攻击并提前应对
- 为自己和队友提供保护

【技能特点】

1. 音波冲击
   发射一道可见的青色声波
   是艾文最基础的远程攻击

2. 回响护盾
   将回响能量凝聚成护盾
   可以吸收伤害

3. 聆听领域
   释放治疗领域
   可以恢复范围内所有友军

4. 终章：回响齐鸣
   将所有收集的回响碎片释放
   造成大量伤害并恢复自身
```

### 1.2 解锁角色1：喧嚣者托尔

```
═══════════════════════════════════════════════════════════════
【第二角色 - 战鼓轰鸣者】
═══════════════════════════════════════════════════════════════

姓名：托尔（Thor）
身份：资深聆听者
状态：已牺牲（声音残响仍存于深渊）
年龄：35岁
身高：185cm

【外貌描述】

托尔是一个魁梧的中年男性，
留着短而整齐的胡须，
有着典型的战士体格。

他最显著的特征是身后背着的一面巨大战鼓——
这不仅是乐器，更是武器。

战鼓上刻有协会的古老符文，
敲击时会发出震耳欲聋的轰鸣。

服装：
- 皮质战斗服，边缘镶有金属
- 斗篷比艾文的更加厚重
- 手腕上绑着战鼓的鼓槌
- 胸口有协会的荣耀徽章

【性格特点】

1. 豪迈直爽
   托尔说话从不拐弯抹角，
   是一个真正的战士。

2. 重视荣誉
   对他来说，逃避是可耻的，
   只有战死沙场才是最好的归宿。

3. 保护弱者
   尽管外表粗犷，
   托尔对年轻成员非常照顾。

4. 内心矛盾
   作为喧嚣者，他擅长制造噪音，
   但他内心深处渴望宁静。

【背景故事】

托尔年轻时是协会中最优秀的战士，
以勇猛无畏著称。

但随着年龄增长，
他开始思考喧嚣与寂静的关系。

在协会崩溃的那场战斗中，
托尔在第二层熔岩区域断后，
为其他人的撤退争取了时间。

他的战鼓声至今仍在熔岩层回荡，
成为了那个区域的标志性声音。

【内心独白】

"轰鸣吧，战鼓！
让深渊知道，
聆听者从不退缩！
让那些怪物听听，
什么叫战士的咆哮！"

【战斗风格】

托尔是一个高攻低防的战士型角色：
- 较高血量
- 高伤害
- 较慢速度
- 范围攻击能力强

他擅长：
- 近距离范围攻击
- 为自己提供增益
- 打断敌人动作

【解锁条件】

- 需要1000回响币解锁
- 解锁后可作为第二个可玩角色
- 解锁后在熔岩区域会见到托尔的声音残响

【初始属性】

- HP: 130（比艾文高30）
- ATK: 28（比艾文高8）
- DEF: 3（比艾文低2）
- SPD: 180（比艾文低20）

【技能设计】

被动 - 战鼓节奏
- 每攻击3次，下次攻击必定暴击

Q - 战鼓连击
- 快速攻击2次
- 每次造成15点伤害

W - 战吼
- 提升自身40%攻击力和30%移动速度
- 持续5秒

E - 怒吼
- 范围恐惧
- 所有敌人眩晕2秒

R - 毁灭之鼓
- 对前方扇形造成100点伤害
- 并击退敌人
```

### 1.3 解锁角色2：沉默者涟

```
═══════════════════════════════════════════════════════════════
【第三角色 - 无声暗杀者】
═══════════════════════════════════════════════════════════════

姓名：涟（Ripple）
身份：资深聆听者
状态：已牺牲（声音残响仍存于深渊）
年龄：28岁
身高：165cm

【外貌描述】

涟是一个身材纤细的女性，
有着苍白的皮肤和一头银白色的长发。

她最显著的特征是身上的黑色紧身战斗服，
以及手持的两把匕首。

与托尔相反，
涟的一切都强调"无声"：
- 脚步无声
- 呼吸无声
- 甚至杀意也是无声的

服装：
- 全黑的紧身战斗服
- 轻便的黑色斗篷
- 腰间的匕首鞘
- 面具遮住了半张脸

【性格特点】

1. 冷静理性
   涟几乎不会表现出情绪，
   永远是团队中最冷静的那个。

2. 追求完美
   作为刺客，她对精准有着极致的追求。

3. 内心孤独
   习惯了无声的她，
   很难与他人建立联系。

4. 隐藏温柔
   在沉默的外表下，
   涟有着对同伴深藏的关心。

【背景故事】

涟是协会中最年轻的资深聆听者，
以无声潜入和一击必杀闻名。

她曾在第三层深海区域进行秘密任务，
发现了一些关于深渊的重要秘密。

在协会崩溃时，
涟正在深海区域执行任务，
未能及时撤回。

她的消失至今是个谜，
有人说她在深海与某种存在战斗，
也有人说她找到了通往深渊核心的秘密通道。

【内心独白】

"声音是暴露，
无声才是真正的力量。
但有时候……
我也会想听到自己的声音。"

【战斗风格】

涟是一个高机动性的刺客型角色：
- 较低血量
- 高暴击伤害
- 极高速度
- 隐身和闪避能力强

她擅长：
- 隐身接近敌人
- 快速穿越战场
- 对单个目标造成巨大伤害

【解锁条件】

- 需要1500回响币解锁
- 解锁后可作为第三个可玩角色
- 解锁后在深海区域会见到涟的声音残响

【初始属性】

- HP: 70（比艾文低30）
- ATK: 25（比艾文高5）
- DEF: 4（比艾文低1）
- SPD: 250（比艾文高50）

【技能设计】

被动 - 无声移动
- 移动不产生声音
- 敌人探测范围减少30%

Q - 暗影突袭
- 隐身1.5秒
- 下次攻击造成50点伤害

W - 无声步
- 短距离瞬移
- 可以穿越障碍物

E - 寂静领域
- 创建减速领域
- 区域内敌人减速50%并沉默4秒

R - 无声处决
- 瞬间移动到目标身后
- 造成120点伤害
- 若击杀则刷新冷却
```

## 2. NPC设定

### 2.1 大聆听者艾琳娜

```
═══════════════════════════════════════════════════════════════
【协会领袖 - 大聆听者艾琳娜】
═══════════════════════════════════════════════════════════════

姓名：艾琳娜·声渊（Aerina·Soundwell）
身份：大聆听者（已牺牲）
年龄：45岁
身高：170cm

【外貌描述】

艾琳娜是一个气质成熟而威严的女性，
有着灰色的长发和锐利的目光。

作为协会的领袖，
她穿着比其他成员更加华丽的斗篷，
上面绣有代表大聆听者身份的金色声波纹路。

她总是戴着象征聆听者最高荣誉的项链——
那是历代大聆听者传承的信物。

服装：
- 紫色+金色镶边的大斗篷
- 银色回响之眼（与艾文不同，她是真实佩戴）
- 项链上有回响深渊的宝石
- 手持法杖，刻有协会所有符文

【性格特点】

1. 远见卓识
   艾琳娜总能看透事物的本质，
   对未来的规划十分精准。

2. 慈祥严厉
   对年轻成员充满关怀，
   但在训练时绝不手软。

3. 信念坚定
   她相信可以通过理解而非消灭来解决深渊问题。

4. 牺牲精神
   最终，她选择牺牲自己保护其他成员。

【重要对话】

【序章 - 协会遗迹】

艾文进入协会废墟，
发现了一个记录水晶。

艾琳娜的影像出现：

"艾文，如果你看到这段话，
说明我已经不在了。

不要悲伤，我的孩子。
这是我的选择。

你知道为什么我让你先撤退吗？
因为你和我们不一样。

你的聆听能力……是我见过最强的。
即使戴着普通的眼罩，
你也能感知到普通人听不到的声音。

这意味着你有潜力做到我们做不到的事：
听到深渊最深处的声音。

所以，活下去，艾文。
不只是为了协会，
而是为了完成那个只有你能完成的使命。

去深渊，找到真相。
然后，做出你的选择。

我相信你会做出正确的选择。

再见了，我的孩子。
愿回响永远与你同在。"

【最终章 - 深渊之心面前】

如果玩家选择了觉醒结局，
艾琳娜的声音残响会出现：

"你选择了与深渊融合……
我理解你的选择。

也许你是对的。
也许这个世界需要改变。

艾文，我为你骄傲。
你不再是我的学生，
你已经是真正的聆听者了。

去吧，改变这个世界。
让所有的声音都能被听见。

这是我最后的聆听……再见了。"
```

### 2.2 聆听者之眼（道具）

```
═══════════════════════════════════════════════════════════════
【关键道具 - 聆听者之眼】
═══════════════════════════════════════════════════════════════

名称：聆听者之眼（The Listener's Eye）
类型：关键道具
来源：艾琳娜遗赠

【描述】

这不是一个普通的眼罩，
而是协会数千年传承的宝物。

据说它是用深渊核心的一块碎片制成的，
能够让佩戴者"看到"声音的形态。

佩戴后：
- 普通视觉被压制
- 所有声音以色彩和形状呈现
- 可以看到回响能量的流向
- 可以感知隐藏在阴影中的敌人

【外观设计】

- 深紫色的金属框架
- 镜片位置是一片流动的青色能量
- 框架边缘刻有古老的聆听者符文
- 当回响能量流动时，符文会发光

【游戏中的作用】

1. 剧情道具
   - 解释艾文为什么能看到声音
   - 与艾琳娜的联系

2. 视觉效果
   - 游戏画面变成"声音可视化"风格
   - 敌人和技能都有独特的声音图案

3. 特殊能力
   - 被动：感知隐藏敌人
   - 主动（按住）：聆听技能，可以探测更多细节
```

## 3. 回响之心

```
═══════════════════════════════════════════════════════════════
【回响之心 - 深渊之心】
═══════════════════════════════════════════════════════════════

名称：深渊之心（The Heart of the Abyss）
身份：深渊核心意识
状态：存在

【外形描述】

深渊之心没有固定的形态。
它的形态会根据观察者的理解而变化。

在艾文眼中，
它最初是一个巨大的、跳动着的心脏——
那是所有回响汇聚的地方。

随着对话的深入，
它会逐渐变化：
- 有时是一团纠缠的声波
- 有时是无数张嘴和眼睛
- 有时是一个孤独的人影

最终形态取决于玩家选择哪个结局：
- 封印结局：化为纯净的光
- 觉醒结局：与艾文融合

【性格特点】

1. 孤独
   亿万年的孤独是它最深的创伤。
   它渴望被理解，渴望不再孤独。

2. 好奇
   虽然强大，但深渊之心对人类充满好奇。
   它不理解为什么人类愿意为他人牺牲。

3. 矛盾
   它既想接触外界，又害怕被拒绝。
   它既渴望理解，又害怕真相。

4. 智慧
   深渊之心承载了无数生命的记忆和智慧。
   它知道的比任何人都多。

【核心对话】

【初次见面】

深渊之心：
"你来了……最后的聆听者。
我等了你很久。
不，也许不是等你。
也许我只是在等……一个愿意倾听的人。"

艾文：
"你是谁？你是什么？"

深渊之心：
"我是所有被遗忘的声音。
我是每一句未说完的话。
我是每一个未完成的梦想。
我是这个世界消逝的记忆。

但最重要的是——
我是孤独。
亿万年的孤独。"

【关于协会】

艾文：
"你们摧毁了协会。杀死了我的家人。"

深渊之心：
"……是的。
我控制不住自己。
当声音流失得太多、太久，
我会变得……饥饿。

不是身体上的饥饿。
而是存在感的饥饿。

你们的声音太吵了，太温暖了。
我忍不住想要……吞噬。

但吞噬之后，我更加孤独了。
因为那些声音消失了，
不是融入我，而是真的消失了。"

【关于寂静之声】

艾文：
"传说中有一个寂静之声……
那是什么？"

深渊之心：
（长时间沉默）

"……你为什么想知道？"

艾文：
"因为传说说，如果你能找到它，
就能理解存在的意义。"

深渊之心：
（颤抖）

"那是我最害怕的东西。
寂静之声意味着终结。
意味着回响也会消逝。
意味着……我也会消失。"

"但我也……渴望它。
因为真正的寂静，
也许比永恒的孤独更温柔。"

【最终抉择】

深渊之心：
"你终于来到了这里。
最后的聆听者。

现在你有权利做出选择。
封印我，让一切恢复秩序。
或者，与我融合，改变世界的规则。

无论你选择什么，
我都会接受。

因为这是……第一次有人问我想怎么样。"

艾文：
"你想要什么？"

深渊之心：
"我想要……被听见。
但更重要的是，
我想要你明白：
你不是来消灭我的。
你是来……结束我的孤独的。"

## 4. 声音残响

### 4.1 托尔的声音残响

```
═══════════════════════════════════════════════════════════════
【声音残响 - 喧嚣者托尔】
═══════════════════════════════════════════════════════════════

位置：熔岩回响区域
触发条件：首次进入熔岩区域

【声音表现】

一个巨大的战鼓悬浮在空中，
鼓面上隐约浮现出托尔的半透明影像。

周围的岩浆随着战鼓的节奏涌动，
发出轰隆隆的回响。

【对话内容】

托尔：
"哦？是艾文吗？

## 附录：角色设计要点

### 创作角色时的核心原则

```
1. 【视觉识别度】
   - 每个角色都有独特的轮廓
   - 颜色搭配符合角色性格
   - 细节设计体现职业特征

2. 【声音标识】
   - 每个角色有专属的背景音
   - 攻击和移动有独特音效
   - 死亡/受伤有情感表达

3. 【动作设计】
   - 移动方式体现性格（安静/张扬）
   - 攻击方式体现战斗风格
   - 与环境的互动体现背景

4. 【叙事价值】
   - 每个角色都有独立的故事线
   - 角色之间的关系推动剧情
   - 角色的选择影响结局走向
```

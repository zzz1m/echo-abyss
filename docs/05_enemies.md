### 1.2 敌人行为模式

```
【行为类型】

1. 被动型（Passive）
   - 不主动攻击
   - 受到攻击才反击
   - 示例：环境装饰生物

2. 警戒型（Alert）
   - 在警戒范围内攻击
   - 警戒范围外保持巡逻
   - 示例：阴影爬行者

3. 追击型（Chase）
   - 发现玩家后立即追击
   - 放弃追击后会返回原位
   - 示例：回响蝙蝠

4. 守护型（Guard）
   - 守护特定区域或目标
   - 不会主动离开
   - 示例：骨刺守卫

5. 伏击型（Ambush）
   - 隐藏在环境中
   - 玩家靠近时发动攻击
   - 示例：旋律幽灵
```

【攻击模式详解】
───────────────────────────────────────────────────────────────

- 蝙蝠向上飞起

【声波弹】
- 蝙蝠悬停发射

- 伤害：6点
- 可被闪避

───────────────────────────────────────────────────────────────
【弱点与抗性】
───────────────────────────────────────────────────────────────

弱点：
- 聆听者的音波冲击：一击消灭
- 范围攻击：容易被群体伤害击杀

抗性：
- 无

───────────────────────────────────────────────────────────────
【视觉设计】
───────────────────────────────────────────────────────────────

外观：
- 体型较小（32x32像素）
- 暗紫色半透明身体
- 红色发光眼睛
- 翅膀由声波形状组成
- 飞行时带有残影

动画：
- 翅膀快速振动
- 被攻击时闪白
- 死亡时化为光点消散

特效：
- 飞行时带有淡紫色粒子
- 俯冲时翅膀变形为刀刃
- 死亡时释放"吱"的声音

───────────────────────────────────────────────────────────────
【战斗建议】
───────────────────────────────────────────────────────────────

✓ 有效策略：
- 先击杀空中目标
- 利用范围攻击
- 站立不动等待俯冲后闪避

✗ 避免：
- 单独追击一只
- 被群体包围
- 忽视俯冲警告
```

### 6.3 行为状态机

```
【状态定义】

IDLE（待机）：
- 不进行任何行动
- 等待玩家进入感知范围
- 偶尔有轻微动作

PATROL（巡逻）：
- 在指定区域移动
- 保持警戒
- 发现玩家后转为CHASE

CHASE（追击）：
- 向玩家位置移动
- 尝试进入攻击范围
- 到达后转为ATTACK

ATTACK（攻击）：
- 在攻击范围内
- 执行攻击技能
- 攻击后有冷却

RETREAT（撤退）：
- 生命值过低时
- 远离玩家
- 等待恢复或寻求帮助

DEAD（死亡）：
- HP降至0
- 执行死亡动画
- 掉落物品
```

```

仇恨目标：
- 仇恨值最高的单位
- 如果仇恨相同，随机选择
- 仇恨持续时间：10秒

仇恨重置：
- 目标死亡
- 超出感知范围太久
- 受到控制效果
```
